# Task 3: To-Do List (Synent Internship)

## 📖 Project Overview
This project is a **command-line To-Do List manager** built in Python.  
It allows users to add, view, and delete tasks interactively.  
The goal is to practice Python fundamentals like lists, loops, and error handling.

---

## ⚙️ Features
- Add new tasks  
- View all tasks  
- Delete tasks by number  
- Menu-driven CLI interface
